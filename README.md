# Gradient Descent for Convex Optimization

This repository contains a collaborative project developed for the *Optimization for Data Science* course during my MSc in Data Science.

The project was developed together with two classmates and focused on solving a multiclass classification problem using optimization techniques based on Gradient Descent methods.

Following the guidelines provided by the course instructor, we implemented the algorithms using the assigned convex loss function and analyzed their performance by monitoring:
- loss reduction across iterations,
- accuracy improvement,
- and computational cost in terms of CPU time.

Since the optimization problem was convex, the algorithms were guaranteed to converge toward progressively lower values of the loss function throughout the iterations.

The implemented algorithms were:
- **Standard Gradient Descent (GD)**
- **Randomized Block Coordinate Gradient Descent (BCGD)**
- **Gauss-Southwell BCGD**

The project also includes a detailed report describing:
- the theoretical background of the optimization methods,
- algorithm implementation details,
- and an analysis of the experimental results and performance comparisons.
